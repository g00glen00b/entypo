Entypo comes in two flavours, choose the icon set that fits your needs

Entypo

Entypo is the name of the original iconset. This set guarantees to be backwards compatible to the original Github project.

View Entypo

Entypo+

Entypo+ is the name of the new iconset, provided by the same creator, Daniel Bruce. It has a lot of new icons, but some icons of the original set were left away.
